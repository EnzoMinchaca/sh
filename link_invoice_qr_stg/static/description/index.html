How to create Custom QR based Invoice into odoo

1. Once you installed QR code show and print module in odoo then app will show

2. Navigate to the invoice section, then access the configuration menu, and proceed to the settings.

3. In the configuration settings, customize the QR code in the customer invoices section and save the changes.

4. navigate to the invoice section and create a new invoice.

5. To create a custom URL for generating a QR code, please select the option to print the QR code. Once you do so, the QR code will be generated.

6. After confirming the invoice, you can navigate to the settings and click on "Print Invoices." This will allow you to print the invoice, which will also include a custom QR code in the corner.

7. When you print the invoice, the custom QR code will be included in the top right corner of the invoice in PDF format.
